# UVL Dynamic

## Introduction

UVL Dynamic adds 2 blocks to the side:
 * Search all items - a link to search all items;
 * Collection menu - a menu of subcollections for the current collection, based on menu menu-sub-collections.

## Installation
 
Install as usual, see [this](https://drupal.org/documentation/install/modules-themes/modules-7) for further information.
